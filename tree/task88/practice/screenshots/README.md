# 实践截图存放目录

请将所有实践操作的截图按照以下命名规范存放在此目录：

## 截图命名规范

### 基础操作截图
- `01-env-check.png` - 环境验证（go version, git version等）
- `02-git-clone.png` - 源码克隆成功
- `03-build-success.png` - 编译成功输出
- `04-geth-version.png` - Geth版本信息
- `05-init-chain.png` - 区块链初始化
- `06-node-start.png` - 节点启动

### 控制台操作截图
- `07-basic-query.png` - 基础查询（区块高度、账户、余额）
- `08-mining-test.png` - 挖矿前后对比
- `09-create-account.png` - 创建新账户

### 交易操作截图
- `10-send-tx.png` - 发送交易
- `11-tx-details.png` - 交易详情
- `12-balance-change.png` - 余额变化确认

### 智能合约截图
- `13-compile-contract.png` - 合约编译
- `14-deploy-contract.png` - 合约部署交易
- `15-contract-address.png` - 合约地址确认
- `16-call-contract.png` - 合约调用
- `17-event-logs.png` - 事件日志

### 外部工具截图
- `18-curl-rpc.png` - curl RPC测试
- `19-web3js-test.png` - Web3.js连接测试
- `20-block-explorer.png` - 区块浏览器（可选）
- `21-block-info.png` - 区块详细信息
- `22-tx-info.png` - 交易详细信息

## 截图要求

1. **清晰度**：确保文字清晰可读
2. **完整性**：包含完整的命令和输出
3. **时间戳**：保留终端的时间戳信息
4. **格式**：推荐使用PNG格式

## 提示

- Windows用户可使用 `Win + Shift + S` 截图
- macOS用户可使用 `Cmd + Shift + 4` 截图
- Linux用户可使用 `gnome-screenshot` 或 `flameshot`

## 快速开始

### 自动化部署（推荐）

**Windows:**
```powershell
cd scripts
.\deploy-geth.ps1
```

**Linux/macOS:**
```bash
cd scripts
chmod +x deploy-geth.sh
./deploy-geth.sh
```

### 手动启动开发链

```bash
# 最简单的方式
./geth --dev --http console

# 或查看自动生成的启动脚本
./start-geth.sh  # 或 start-geth.ps1
```
